GGHub

https://gghub.com.br

Software as a Service (SaaS)

A GGHub é uma plataforma de gerenciamento de torneios de jogos online. Nela você pode comprar fichas para participar de torneios, acompanhar os resultados e rankings, além de poder criar torneios personalizados para jogar com outros oponentes e ganhar dinheiro com isso.

O sistema foi feito com as tecnologias do ReactJs, Laravel, Mysql.
