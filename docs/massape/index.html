Massapê Imóveis

https://massapeimoveis.com.br/

Software as a Service (SaaS)

A Massapê Imóveis foi um site para venda de imóveis que trabalhei no front-end. Nele você pode encontrar imóveis para alugar e comprar, além de poder entrar em contato com a empresa ou proprietário do imóvel para tirar dúvidas e agendar visitas.

O sistema foi feito com as tecnologias do ReactJs, Laravel e Mysql.
